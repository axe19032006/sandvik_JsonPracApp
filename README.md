# Sandvik Proposal App

A Streamlit application for generating Sandvik project proposals and analyzing vendor bids.

## Setup

1. Create a virtual environment: `python -m venv .venv`
2. Activate it: `.venv\Scripts\activate` (Windows) or `source .venv/bin/activate` (Mac)
3. Install dependencies: `pip install streamlit`
4. Run: `streamlit run app.py`"# sandvik_JsonPracApp"
